Comments

The software is quite intuitive and easy to use. There is a zipped data file that can be downloaded that includes the drawings for all figures in the paper which is ideal. However, there seems to be some parameter fiddling to obtain the results. For instance, in Fig. 11, I had to specify a "min region size" of 8 instead of the default 7 to reproduce the two rightmost images (and this parameter set also works for the two leftmost images).
